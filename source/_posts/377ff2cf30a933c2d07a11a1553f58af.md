## On Arch Linux
```bash
pacman -S putty
```
Then cd to path of key(s):
```bash
puttygen keyname -o keyname.ppk
```

## On Debian/Ubuntu
```bash
apt-get install putty-tools
puttygen keyname -o keyname.ppk
```

## Notice!
replace the “keyname” with your private key filename!
