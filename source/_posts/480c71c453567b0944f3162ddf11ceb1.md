## 清空所有数据
```bash
python manage.py flush
```


## 重建表结构
1. 删除数据库中所有表
2. 删除应用文件夹下的`migrations`目录内所有文件，除了`__init__.py`
3. 执行以下命令
```bash
python manage.py makemigrations
python manage.py migrate
```
