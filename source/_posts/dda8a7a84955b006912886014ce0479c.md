## 操作步骤
我用的是Ubuntu系统，首先需要安装qemu-kvm：
```bash
sudo apt install qemu-kvm
```

将当前用户添加到kvm用户组：
```bash
sudo adduser $USER kvm
```

检查`/dev/kvm`所有者：
```bash
ls -al /dev/kvm
```
一般结果就是所有者是root以及kvm这个组

添加完了之后检查一下kvm组里有没有你的用户名：
```bash
grep kvm /etc/group
```

一般结果就是这样：
```
kvm:x:数字:用户名
```

之后重启系统生效，如果不想重启可以运行以下命令(仍然得注销后再登录才生效)：
```bash
udevadm control --reload-rules && udevadm trigger
```

ps：反正我选择重启
